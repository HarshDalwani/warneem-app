# warneem-app

## Build Setup

```bash
# clone git repo 
$ git clone https://github.com/HarshDalwani/warneem-app.git

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
